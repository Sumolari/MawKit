Detailed Description

GameKit namespace offers a secure wrapper for achievements and leaderboards backed by Game Center on Apple devices and Google Play Services on Android.

Type of callbacks called after finishing an asynchronous request.

Receives as first argument value retrieved and as second argument whether value was successfully retrieved (true) or not. Value unsuccessfully retrieved has no meaning and should not be used. Initializes internal data structures, local player and so on.
