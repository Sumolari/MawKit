Detailed Description

template<typename T, typename = typename std::enable_if<std::is_arithmetic<T>::value, T>::type>
class MK::Range< T, typename >

A range defines a mathematical closed range starting at a value a and ending at a value b, both of them included in the range, mathematically expressed as \(\left [a,b\right ]\).

Any value \(a\leq x\leq b\) is considered to be included in the range.
